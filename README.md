# Zeke Smith

Personal site — a hub for projects, ideas, and writing. Built with Next.js, TypeScript, and Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

Projects and articles live in `src/content/`:

- `site.ts` — name, hero, about, nav
- `projects.ts` — project cards
- `articles.ts` — writing drafts

Add new entries there; the homepage picks them up automatically.

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint
